# music-x

![image](https://github.com/samvat7/music-x/assets/108258653/5d8c6416-fc1b-47f4-ba88-5e71c7c98442)


This is a front-end project that includes a navigation bar, search bar, user image, and various cards displaying information about music artists. It also includes Bootstrap and Font Awesome libraries for styling and icons. The project uses responsive design and showcases a list of similar artists and recently played songs.
